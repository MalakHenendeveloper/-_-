\\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"findings\": [...]}'\n`\n\n### Use FormData When:\n- ✅ File uploads required\n- ✅ Browser form submissions\n- ✅ Mixed text + binary data\n- ✅ body: formData\n});\n`\n\n---\n\n## ⚠️ Common Mistakes\n\n### ❌ Mistake 1: Not stringifying findings in FormData\n`javascript\n// WRONG ❌\nformData.append('findings', findings);  // Object/Array
